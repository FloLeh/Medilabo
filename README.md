## Pratiques à suivre pour respecter la démarche Green Code

- Optimiser le code pour éviter les calculs et les requêtes inutiles
- Pas de données inutiles dans les BDD
- Architecture légère et scalable pour éviter de surcharger des serveurs
- Pas de dépendances inutiles dans les microservices (à voir l’impact de Lombok)
- Docker avec des images “légères”
- Éviter l’abus de logs
- Utilisation de caches (pour les données / docker)
